# Portable Karaoke Machine (Python)

This project is a Python-based desktop karaoke system that allows users
to search and play YouTube karaoke videos.

## Features
- YouTube karaoke search
- Video playback via VLC
- Simple desktop interface
- Designed as a demo / portfolio project

## Environment
- Python 3.10
- Windows 10 / 11
- VLC Media Player installed

## How to Run
```bash
python KTV_demo.py
